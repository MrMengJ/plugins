## nav/jQuery-topnav（jquery插件）
### 导航栏自动固定顶部

body底部加入 script 脚本即可

```
    <script src="js/jquery.topnav.js"></script>

    <script>
        /*假如导航栏的id值为nav，只要将其改为自己对应的选择器则行*/
        $('#nav').navFixed();    
    </script>
```
